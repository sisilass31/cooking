# Cooking website

## Présentation

Recettes Délices est un site web vitrine conçu pour mettre en avant une sélection de recettes de cuisine, de manière élégante et visuellement attrayante. Le site présente une collection soigneusement choisie de recettes, chacune accompagnée de belles images et d'instructions claires pour inspirer les amateurs de cuisine à essayer de nouveaux plats.
